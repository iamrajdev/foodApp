
# Recipe Food App :man_cook:
Simple food recipe app. It show all food recipe list and details information.

## API
-[themealdb](https://www.themealdb.com/)

## App Features :dart:

- Get random meal
- Get popular meal
- Meal save to favorite database
- Delete favorite meal
- Undo favorite meal
- Get category meal
- Search meal by name
- Loading view
